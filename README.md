# Todo App — Kotlin + React

Fullstack todo-приложение с REST API на Spring Boot (Kotlin) и фронтендом на React. Всё запускается через Docker Compose одной командой.

## Стек

**Backend**
- Kotlin + Spring Boot
- PostgreSQL 16

**Frontend**
- React 19 
- CSS

**Контейнеризация**
- Docker

## Возможности

- Создание, редактирование и удаление задач
- Отметка задачи как выполненной
- Детальная страница каждой задачи
- Пагинация списка (9 задач на страницу)
- Обработка ошибок на фронтенде и бэкенде
- Валидация входных данных (backend)

## API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/todos` | Получить все задачи |
| GET | `/todos/{id}` | Получить задачу по ID |
| POST | `/todos` | Создать задачу |
| PUT | `/todos/{id}` | Обновить задачу |
| DELETE | `/todos/{id}` | Удалить задачу |
